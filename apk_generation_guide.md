# AuraFit PWA-ни Андроид APK Файлига Айлантириш Йўриқномаси

Ушбу йўриқномада **AuraFit** Progressive Web App (PWA) иловасини профессионал кўринишдаги, телефонга ўрнатиладиган ва Google Play Store-га чиқаришга тайёр Андроид **APK** форматига ўтказишнинг 2 та энг самарали усули келтирилган.

---

## 🚀 1-усул: PWABuilder (Энг тез ва тавсия этиладиган усул)

**Microsoft PWABuilder** — PWA иловаларни бир неча сония ичида Google Play учун тайёр Android WebAPK (TWA) пакетига айлантириб берувчи бепул расмий онлайн хизматдир.

### 📝 Бажарилиш босқичлари:
1. **Иловани Интернетга жойланг:**
   Лойиҳа папкасини (`Калория бот`) хоҳлаган HTTPS протоколи билан ишловчи бепул хостингга (масалан, [Vercel](https://vercel.com/) ёки [Netlify](https://www.netlify.com/)) юкланг.
   * *Масалан:* Иловангизнинг оммавий манзили `https://aurafit.vercel.app` кўринишида бўлсин.
2. **PWABuilder-га киринг:**
   Браузер орқали [www.pwabuilder.com](https://www.pwabuilder.com/) сайтига киринг.
3. **Ҳаволани киритинг:**
   Илова ҳаволасини сайтдаги махсус қидирув майдонига ёзинг ва **"Start"** тугмасини босинг.
4. **Таҳлил натижаси:**
   Тизим иловангизнинг `manifest.json` ва `sw.js` (Service Worker) файлларини текширади. AuraFit иловасининг кўрсаткичлари 100% тайёр бўлгани учун тизим сизни автоматик равишда юклаб олиш ойнасига ўтказади.
5. **APK-ни юклаб олинг:**
   **"Build My APK"** (ёки "Generate") тугмасини босинг. Пайдо бўлган созламалардан:
   * Илова пакети номи: `com.elyor.aurafit`
   * Илова номи: `AuraFit`
   * **"Download APK"** тугмасини босинг ва бир неча сония кутинг.
6. **Натижа:**
   Сизга `.zip` архив юклаб берилади. Архив ичидан телефонингизга ўрнатиш учун тайёр **`app-release-signed.apk`** файли ҳамда Google Play Store-га жойлаш учун керакли имзо калитлари (`keystore`) ўрин олган бўлади.

---

## 🏋️‍♂️ 2-усул: Capacitor (Иловани ички Веб-Ойнада маҳаллий ўраш)

Агар иловани ҳеч қандай сайтга жойламасдан, тўғридан-тўғри ўз компьютерингизда маҳаллий (offline) пакетловчи Андроид илова қилмоқчи бўлсангиз, **Capacitor** тизимидан фойдаланамиз.

### 📝 Бажарилиш босқичлари:

#### 1. Керакли дастурларни тайёрлаш:
* Компьютерингизда [Node.js](https://nodejs.org/) ўрнатилган бўлиши лозим.
* [Android Studio](https://developer.android.com/studio) дастури ва Android SDK ўрнатилган бўлиши керак.

#### 2. Capacitor-ни лойиҳага қўшиш:
Компьютерингиз терминалини (PowerShell) очиб, AuraFit иловаси жойлашган папкага киринг ва қуйидаги буйруқларни бажаринг:

```bash
# 1. Capacitor кутубхоналарини ўрнатиш
npm install @capacitor/core @capacitor/cli

# 2. Иловани тизимга таништириш
# Бу ерда илова номи: AuraFit, пакет ID-си: com.elyor.aurafit деб белгиланади.
npx cap init AuraFit com.elyor.aurafit --web-dir=.

# 3. Android платформасини лойиҳага қўшиш
npm install @capacitor/android
npx cap add android
```

#### 3. Андроид иловани синк қилиш ва кодни тайёрлаш:
Лойиҳангиздаги файллар ўзгарганидан сўнг, уларни Android папкаси билан бирлаштириш учун:
```bash
npx cap sync
```

#### 4. Android Studio орқали APK олиш:
1. Қуйидаги буйруқ орқали Android Studio-ни очган ҳолда лойиҳани ишга туширинг:
   ```bash
   npx cap open android
   ```
2. Android Studio лойиҳани тўлиқ юклаб олганидан сўнг (Gradle қуриш тугагач), юқоридаги менюдан:
   * **Build** -> **Build Bundle(s) / APK(s)** -> **Build APK(s)** тугмасини босинг.
3. Бир неча сониядан сўнг экраннинг ўнг пастки бурчагида **"APK(s) generated successfully"** ёзуви чиқади. **"Locate"** тугмасини босиб, тайёр бўлган `app-debug.apk` файлини олишингиз ва исталган Андроид телефонга ўрнатиб ишлатишингиз мумкин!

---

> [!TIP]
> **Тез ва муаммосиз натижа учун 1-усулни (PWABuilder) танлашни тавсия этаман.** Чунки у иловага ҳеч қандай қўшимча оғир кутубхоналар қўшмасдан, иловани жуда енгил ва тезкор (атиги 2-3 MB ҳажмда) APK файл қилиб беради!
